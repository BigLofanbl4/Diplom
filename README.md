# Diplom

## Mock API for frontend

The frontend requests API under `/api/v1` (through Vite proxy to `http://127.0.0.1:8000`).

1. Start mock backend:
```bash
npm run mock
```
2. In another terminal start frontend:
```bash
npm run dev
```

Mock auth credentials:
- `admin` / `admin123`
- `teacher` / `teacher123`
